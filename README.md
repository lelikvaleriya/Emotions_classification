# Emotions_classification
Проведена оценка качества распознавания эмоций для двух моделей. Проведен анализ эмоций в ответ на два предъявляемых рекламных ролика.

## Задание 1. Оценить качество классификации эмоций нейронной сетью.
Были разработаны две (нейросети) для распознавания эмоций. Для проверки качества классификации был спланирован тест: записано 84 видео, на которых люди изображали одну из 7 эмоций - anger, happy, surprise, sad, fear, disgust или neutral (нейтральное состояние). Если эмоция, которую распознала нейросеть для конкретного видео, совпадала с изначальным лейблом, классификация считалась корректной. Если же распознанная эмоция отличалась от истинной, результат считался ошибочным. Проведена оценка качества двух моделей и выбрана лучшая.

## Задание 2. Оценить эмоциональный отклик зрителей на два рекламных ролика
Рассчитаны эмоциональные сдвиги относительно нейтрального состояния для каждого респондента и каждой эмоции. Проведена аггрегированная оценка эмоционального отклика для обоих
роликов. Визуализирован результат, сравнены два ролика в контексте их влияния на эмоции зрителей. Проведен статистический анализ значимости различий.
